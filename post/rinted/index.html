rinted 做端口转发

2020-02-15
linux
约 118 字 预计阅读 1 分钟

文章目录

    • 安装 rinted
    • 配置

因为腾讯云的 Mongo 和 Redis 都是内网的地址，不能访问，官方说内网进行内网转发就行，于是就用 rinted 做了一个内网转发。

安装 rinted

由于是 Debian 系统，源里有就可以直接安装。

apt install rinted

配置

配置文件也很简单，按照如下个格式配置就好了。

# bindadress    bindport  connectaddress  connectport

成功之后用 Mongo 连一下，畅通无阻。
